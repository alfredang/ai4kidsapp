# Changelog

All notable changes to the AI4Kids iOS/iPadOS app. The newest entry's bullets
are the source of truth for the App Store "What's New" notes.

## [1.2] — 2026-07-21 · build 6

### Added
- Parental gate: a grown-ups-only math challenge now guards the Parents' Corner (which holds the progress reset), matching the Android app's Families Policy gate.

### App Store submission
- Submitted 2026-07-21. Fully offline, no data collection ("Data Not Collected" privacy label unchanged).

## [1.1] — 2026 · build 5 (released)

### Added
- Brain Arcade replaced Brain Games: eight quick solo card games (matching, memory, sums and sorting), fully offline.

## [1.0] — 2026 · builds 1–3

### Added
- Initial release: Phonics Playground, Story Builder, Code Puzzles, Brain Games; universal (iPhone + iPad); stars progress with Parents' Corner reset.
